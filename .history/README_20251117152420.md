# 🎵 Music Text Generator

App che isola la voce da un brano musicale e genera testo partendo dai suoni vocali, usando AI per la separazione vocale, trascrizione e generazione creativa.

## 🚀 Funzionalità

- **Separazione Vocale**: Isola la traccia vocale dagli strumenti usando Demucs
- **Trascrizione**: Analizza la voce con Whisper (speech-to-text)
- **Generazione Testo**: Trasforma fonemi e suoni vocali in testo creativo usando NLP
- **Interfaccia Web**: Frontend React/Next.js per upload e visualizzazione
- **Export**: Scarica testo generato e traccia vocale isolata

## 📋 Requisiti

- Python 3.8+
- Node.js 18+
- (Opzionale) OpenAI API Key per generazione testo avanzata

## 🛠️ Installazione

### Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Su Windows: venv\Scripts\activate
pip install -r requirements.txt
```

**Nota**: L'installazione di Demucs e Whisper può richiedere tempo e spazio (modelli AI).

### Frontend

```bash
cd frontend
npm install
```

## ⚙️ Configurazione

### Backend

Crea un file `.env` nella cartella `backend` e aggiungi:

```
OPENAI_API_KEY=your_key_here
```

**📖 Guida completa**: Vedi [CONFIGURA_OPENAI.md](CONFIGURA_OPENAI.md) per istruzioni dettagliate.

> **Nota**: L'app funziona anche senza OpenAI API Key, ma i risultati sono più basici. Con OpenAI ottieni testi più creativi e professionali!

### Frontend

Crea un file `.env.local` nella cartella `frontend`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## 🏃 Avvio

### 1. Avvia il Backend

```bash
cd backend
python main.py
```

Il server sarà disponibile su `http://localhost:8000`

### 2. Avvia il Frontend

```bash
cd frontend
npm run dev
```

L'app sarà disponibile su `http://localhost:3000`

## 📖 Utilizzo

1. Apri il browser su `http://localhost:3000`
2. Carica un file audio (MP3, WAV, M4A, FLAC)
3. Clicca "Processa Audio"
4. Attendi il processamento (può richiedere alcuni minuti)
5. Visualizza i risultati:
   - Traccia vocale isolata (riproducibile)
   - Trascrizione grezza
   - Testo finale generato
6. Scarica testo e audio vocale

## 🔄 Pipeline AI

1. **Input**: File audio (MP3, WAV, etc.)
2. **Separazione**: Demucs → Voce isolata
3. **Trascrizione**: Whisper → Testo grezzo / Fonemi
4. **Generazione**: NLP (GPT) → Testo finale creativo
5. **Output**: Voce isolata + Testo finale

## ⚠️ Limiti e Note

- **Qualità separazione**: Dipende dal mix originale del brano
- **Accuratezza trascrizione**: Whisper è ottimo sul parlato, meno preciso sul cantato
- **Performance**: Il processamento può richiedere tempo su file lunghi
- **Modelli AI**: Demucs e Whisper scaricano modelli grandi al primo utilizzo
- **OpenAI API**: Opzionale ma consigliato per generazione testo di qualità

## 🐛 Troubleshooting

### Demucs non disponibile
Se Demucs non si installa correttamente, il sistema userà un metodo fallback (estrazione canale centrale).

### Whisper lento
I modelli più grandi (medium, large) sono più accurati ma più lenti. Usa "base" o "small" per test rapidi.

### Errori CORS
Assicurati che il frontend punti al backend corretto in `.env.local`.
