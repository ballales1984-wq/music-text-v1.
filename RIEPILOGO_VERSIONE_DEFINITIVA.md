## 🚀 Come Avviare

### Metodo 1: Collegamento Desktop
1. Doppio click su "Music Text Generator" sul desktop
2. Attendi 5-10 secondi
3. Browser si apre automaticamente

### Metodo 2: Script
1. Doppio click su `avvia_app.bat`
2. Attendi
3. Browser si apre

### Metodo 3: Manuale
```bash
# Terminale 1
cd backend
venv\Scripts\activate
python main.py

# Terminale 2
cd frontend
npm run dev
```
